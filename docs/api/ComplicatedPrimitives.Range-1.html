Translate(Func<T, T>)

Translates (moves) limits of this range by a given translation, preserving their Side abd Type.

Declaration
public Range<T> Translate(Func<T, T> translation)
Parameters
Type Name Description
System.Func<T, T> translation

Function that translates range's limit value.
Returns
Type Description
Range<T>

Range with both limits translated using given translation.
Remarks

This transformation preserves limits' sides which means that it can potentially cause errors if translated values violate any rules applied when creating instance of Range<T>. E.g. the following code:

var range = new Range<int>(0, 10);
var result = range.Translate(e => e * -1);

will throw exception, because left limit value, after translation will be greater than the right limit value.
