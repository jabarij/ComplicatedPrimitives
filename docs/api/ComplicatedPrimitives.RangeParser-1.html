Parse(String)

Converts the specified string to an instance of Range<T>.

Declaration
public Range<T> Parse(string str)
Parameters
Type Name Description
System.String str

A string that contains a string representation of Range<T> as follows:

{ left-limit-type }{ left-limit-value }{ separator }{ right-limit-value }{ right-limit-type }

Where:

  • left-limit-typestring that represents left limit type: '(' for Open, '[' for Closed;
  • left-limit-valuestring that represents left limit value parsed using parser object specified when creating this instance of RangeParser<T>;
  • separatorstring that represents range value separator;
  • right-limit-valuestring that represents right limit value parsed using parser object specified when creating this instance of RangeParser<T>;
  • right-limit-typestring that represents right limit type: ')' for open, ']' for closed;
Returns
Type Description
Range<T>

An instance of Range<T> that is equivalent to the given str.
Exceptions
Type Condition
System.ArgumentNullException

str is either null or empty string
ParsingException

str is not a valid representation of a Range<T>
